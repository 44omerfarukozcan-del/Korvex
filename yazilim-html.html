HTML Eğitimi

HTML, bir web sayfasının iskeletini oluşturan temel yapıdır. Etiketlerle çalışır ve tarayıcıya "bu nedir?" bilgisini verir.

1. HTML Yapısı

Her HTML dosyası temel olarak aşağıdaki yapıyla başlar:

<!DOCTYPE html> <html lang="tr"> <head> <meta charset="UTF-8"> <title>Başlık</title> </head> <body> Buraya sayfa içeriği gelir. </body> </html>

2. En Çok Kullanılan Etiketler

<h1> - <h6> → Başlık etiketleri

<p> → Paragraf

<a href=""> → Link

<img src=""> → Görsel

<div> → Bölüm / kapsayıcı

<h1>Merhaba Dünya</h1> <p>Bu bir paragraftır.</p> <a href="https://korvex.com">Korvex'e git</a> <img src="logo.png" width="150">

3. Liste Yapıları

<ul> <li>Madde 1</li> <li>Madde 2</li> </ul> <ol> <li>Sıralı 1</li> <li>Sıralı 2</li> </ol>

4. Basit Bir HTML Şablonu

<div class="kart"> <img src="urun.jpg"> <h3>Ürün Adı</h3> <p>Kısa açıklama.</p> <a href="#">Satın Al</a> </div>
