CSS Eğitimi

CSS, web sayfalarının görünümünü ve tasarımını kontrol eder. Kutu yapısı, renkler, konumlandırma, responsive tasarım gibi tüm görsel düzenlemeleri CSS ile yaparız.

1. CSS Nedir?

CSS, HTML içeriğine stil eklemeyi sağlar. Renk, boyut, hizalama, sayfa düzeni gibi tüm görsel ayarlar buradan yapılır.

p { color: white; font-size: 18px; }

2. CSS Kullanım Yöntemleri

3 yöntem vardır:

• Inline CSS
• Internal CSS
• External CSS (önerilen)

/* Harici CSS Örneği */ body { background: #0b0d13; color: #fff; }

3. Kutu Modeli (Box Model)

CSS’de her element bir kutudur: margin → border → padding → content

.kutu { width: 200px; padding: 20px; margin: 10px; background: #3bafff; border-radius: 10px; }

4. Flexbox

Öğeleri yatay-dikey hizalamamızı sağlar. Modern tasarımların temelidir.

.container { display: flex; justify-content: center; align-items: center; gap: 20px; }

5. Responsive Tasarım

Mobil uyumluluk için media query kullanılır.

@media(max-width: 768px) { .kutu { width: 100%; } }
