Python Eğitimi

Python, öğrenmesi kolay ve çok güçlü bir programlama dilidir. Web geliştirme, veri bilimi, yapay zekâ, otomasyon ve daha birçok alanda kullanılır.

1. Python’da Değişkenler

Python’da değişken oluşturmak çok basittir. Tür belirtmen gerekmez.

isim = "Ömer" yas = 20 aktif = True print(isim, yas, aktif)

2. Listeler

Birden fazla değeri saklamak için liste kullanılır.

liste = ["HTML", "CSS", "JavaScript", "Python"] print(liste[0]) # HTML print(len(liste)) # 4

3. Fonksiyonlar

Fonksiyonlar tekrar eden işlemleri kolaylaştırır.

def selam(isim): print("Merhaba", isim) selam("Ömer")

4. If — Else Yapısı

Koşullu yapılar karar vermemizi sağlar.

sayi = 15 if sayi > 10: print("Sayı 10'dan büyük") else: print("Sayı küçük")

5. Döngüler

For döngüsü liste üzerinde tekrarlama yapar.

for dil in ["HTML", "CSS", "JS"]: print("Öğren:", dil)
