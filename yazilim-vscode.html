Visual Studio Code Eğitimi

VS Code, yazılım geliştiricilerinin en çok kullandığı modern ve hafif bir kod editörüdür. Eklentiler, temalar, kısayollar ve özellikler sayesinde çok güçlü bir geliştirme ortamı sunar.

1. VS Code Nedir?

Microsoft tarafından geliştirilen ücretsiz bir editördür. HTML, CSS, JavaScript, Python, C#, Java ve daha birçok dili destekler.

2. En Önemli Eklentiler

VS Code’u daha güçlü yapan şey eklentilerdir. En çok kullanılanlar:

• Live Server — HTML dosyalarını canlı çalıştırma • Prettier — Otomatik kod düzenleme • Material Icon Theme — Dosya ikonları • Python — Python geliştirme aracı • GitLens — Git geçmişi görüntüleme

3. Kısayollar (Çok Önemli)

CTRL + B → Sol paneli aç/kapa CTRL + S → Kaydet CTRL + K + C → Seçili alanı yoruma al CTRL + SHIFT + P → Komut paleti ALT + YUKARI/AŞAĞI → Satır taşı

4. Tema ve Görünüm Ayarları

VS Code istediğin gibi özelleştirilebilir. En popüler temalar:

• Ayu • Dracula • One Dark Pro • Tokyo Night • Night Owl

5. Live Server Kullanımı

Web geliştiriciler için en faydalı eklentidir.

• HTML dosyasına sağ tık → "Open with Live Server" • Tarayıcıda anında güncelleme sağlar
